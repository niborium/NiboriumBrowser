# NiboriumBrowser (CefSharp.WinForms)

NiboriumBrowser är en Chromium-webbläsare skapad med CefSharp.WinForms
Stöd för: Gå tillbaka, Gå Framåt, Refresh, adressfält, Go, New Tab/tabs (för skapa flera flikar)
