# NiboriumBrowser (CefSharp.WinForms)

NiboriumBrowser is a Chromium browser built with CefSharp.WinForms\
Support for: Go Back, Go Forward, Refresh, address bar, Go, New Tab/tabs (for creating multiple tabs)
